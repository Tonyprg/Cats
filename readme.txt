Используемые библиотеки:
    time
    random
    requests
    flask
    sqlalchemy
    telebot
    pgzero

Проект состоит из 3-х программ:

    1) game.py - запускает окно и общается с сервером.

    2) server.py - запускает сервер, который считывает с базы данных
           позиции игроков и кол-во собранных ими монет.

    3) bot.py - запускает бота в телеграмме
           У бота cледующие команды:
               /join - создает нового играка.
               /coins - показывает кол-собранных монет.
               left   - движение игрока влево
               right  - движение игрока вправо
               up     - движение игрока вверх
               down   - движение игрока вниз

В проекте присутствует файл settings.txt, в котором содержаться настройки.
Настройки представляют из себя python-словарь, у которого тип ключей и
значений соответствует str. В этом файле можно указать значения для
токена телеграмм бота, хоста и порта сервера.

Обработчики сервера:
    /leaderboard - показывает список лидеров.
    /my_skils_in_AI - показывает мой опыт в сфере ИИ.
